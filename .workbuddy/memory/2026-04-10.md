## 故障排查：网站登录500错误

- **现象**：https://mka-online.cn 显示 500 Internal Server Error，无法登录
- **排查过程**：
  1. PM2进程在线（online），但重启次数108次
  2. 后端 localhost:3000 API 测试正常（返回200）
  3. Nginx 错误日志显示 "rewrite or internal redirection cycle"
  4. 检查 dist 目录：`/var/www/geomanger/frontend/dist/` 不存在！
- **根本原因**：服务器上前端 dist 目录丢失，Nginx try_files 找不到 index.html，死循环返回500
- **修复方法**：本地重新 vite build 构建，SFTP 上传 dist 到服务器
- **结果**：/login 和 / 均恢复 200，网站正常
