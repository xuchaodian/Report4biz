# 2026-04-09 工作日志

## 添加门店图钉预览 (MapView.vue)
- 点击工具箱「添加门店」后，点击地图会先在点击位置显示红色 SVG 图钉
- 图钉带弹入动画（cubic-bezier 弹性效果）
- 弹出添加门店对话框，取消/关闭时自动移除图钉
- 确认提交成功后也移除图钉（真实门店图标接替）
- `addMarkerPin` 变量持有预览 marker，通过 `removeAddMarkerPin()` 清除
- 对话框新增 `@close="onMarkerDialogClose"` 事件处理取消场景
- 对话框加上 `draggable` 属性，支持拖动避免遮盖地图点击位置

## 地址解析功能 (DataView.vue)
- 后端：`POST /api/poi/batch-geocode`，逐条调用高德地理编码接口（30ms间隔防限流）
- 前端：「地址解析」按钮放在「添加门店」左侧
- 流程：上传含 address 列的 CSV → PapaParse 解析 → 批量调用高德 API → 结果展示
- 解析结果支持：导出为 CSV / 直接导入到门店库
- 原始 CSV 的其他字段（品牌/类型/经理等）在导入时一并写入门店记录
