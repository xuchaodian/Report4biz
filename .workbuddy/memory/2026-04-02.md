## 腾讯地图Y轴转换修复

### 问题
Leaflet加载腾讯地图瓦片时出现403拒绝、瓦片全黑/空白等问题。

### 原因
腾讯地图使用TMS坐标系，Y轴计算公式为：`y = 2^z - 1 - leaflet_y`
之前的代码只设置了 `yx: true`，但这不够。

### 解决方案
创建自定义 `L.TencentTileLayer` 类处理Y轴转换：

```javascript
L.TencentTileLayer = L.TileLayer.extend({
  getTileUrl: function(tilePoint) {
    const z = tilePoint.z
    const x = tilePoint.x
    const y = Math.pow(2, z) - 1 - tilePoint.y  // 关键转换
    const s = this._getSubdomain(tilePoint)
    return this._url
      .replace('{s}', s)
      .replace('{z}', z)
      .replace('{x}', x)
      .replace('{y}', y)
  }
})
```
