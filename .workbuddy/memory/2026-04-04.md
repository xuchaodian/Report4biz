## 半径圆检索修复 (20:37)

### 问题描述
- 用户输入半径"2"后点击搜索，地图只显示50米的橙色临时圆，不显示2000米的紫色虚线大圆
- 检索结果没有出现

### 根本原因
1. **Promise 处理逻辑错误**：`ElMessageBox.prompt()` 的 `.then()` 返回值没有正确传递给外层 Promise
2. **`ElMessage.loadingClose()` 函数不存在**：导致 API 调用失败，结果无法显示

### 修复内容
1. 将 `ElMessageBox.prompt()` 改为 `try/catch` + `await` 语法正确获取返回值
2. 将 `ElMessage.loadingClose()` 改为 `loadingMsg.close()`
3. 输入半径后立即绘制正确的紫色虚线搜索圆

## 多边形检索修复 (20:42)

### 问题描述
- 用户点击"多边形检索"后，在地图上绘制多边形但无法闭合，导致无法检索

### 根本原因
1. **坐标格式不匹配**：后端期望数组格式 `[{lng, lat}, ...]`，但前端发送的是字符串格式 `lng,lat;lng,lat;...`
2. **loadingClose 函数不存在**：同样导致 API 调用失败

### 修复内容
1. 前端 `finishPolygonSearch()` 函数中将 `coordinates` 从字符串改为数组格式
2. 修复 `ElMessage.loadingClose()` 为 `loadingMsg.close()`
